## Installation
Add this line to your application's Gemfile:

```ruby
gem "controlled_vocabulary"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install controlled_vocabulary
```

```bash
bundle exec rails controlled_vocabulary:install:migrations
```
